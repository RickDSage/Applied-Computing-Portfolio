## Assessment Brief (Context Only)

the Excel project in the parent directory.

The brief outlines the academic requirements for demonstrating:
- Use of Excel formulas (including VLOOKUP and IF statements)
- Data processing and logical decision-making
- Documentation of methodology and results

📌 **Note:**  
The brief is included for context only.  
The actual implementation, analysis, and documentation are contained in the
main Excel project files.
