# Operating Systems & Command-Line Fundamentals

This section demonstrates applied coursework focused on operating system concepts and practical command-line usage in both Linux and Windows environments.

The work emphasises understanding how operating systems manage files, directories, and processes, alongside hands-on use of command-line tools to perform common administrative tasks.

## Key Areas Covered

- Linux command-line navigation and file system operations
- Windows command-line fundamentals
- File creation, modification, movement, and deletion
- Understanding command syntax, flags, and outputs
- Applying OS concepts to real-world administrative scenarios

## Skills Demonstrated

- Confident use of command-line interfaces
- Logical problem-solving using OS tools
- Attention to accuracy and safe file handling
- Ability to explain technical actions clearly in written documentation

## Notes

This folder contains coursework-based demonstrations using simulated data and environments.  
It focuses on **command-line proficiency and OS fundamentals**, not advanced scripting or automation.
